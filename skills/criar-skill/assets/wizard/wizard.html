Definição da Skill

O que essa skill faz? Descreva o básico.

Sem espaços, sem maiúsculas, sem acentos. Ex: gerar-relatorio
Cada passo = 1 ação. Use "SE [condição] → [ação]" para condicionais.
Dica para workflows complexos
Se sua skill envolve passos detalhados — como acessar um site específico, clicar em botões na interface, preencher formulários, chamar rotas de API, ou navegar entre telas — escreva os passos de forma mais explicativa aqui (ex: "Acessar portal X, clicar no menu Y, preencher campo Z com o CNPJ"). Quanto mais detalhe, melhor o agente executa.

Não precisa ficar perfeito agora. O caminho real é: gere a skill → rode no agente → ajuste o que não funcionou. Cada vez que você corrige um passo no SKILL.md, a skill fica mais precisa. Skills complexas geralmente precisam de 2-3 rodadas de refinamento até ficarem sólidas.
