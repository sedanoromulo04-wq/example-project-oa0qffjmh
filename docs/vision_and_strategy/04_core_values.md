# 04. Valores Nucleares (Core Values) do Sistema

## 1. Elegância (A Forma da Manifestação)
A elegância não é o excesso, mas sim a subtração perfeita. A manifestação se dá por meio de:
- **Design Mínimo:** A interface visual para o founder no OS deve ser intuitiva, sem botões inúteis. Nos outputs (Carrosséis, posts), prevalece a tipografia limpa, e templates fixos sem variações aleatórias que comprometam o tom.
- **Postura Intelectual:** Sem clamar atenção barata, a forma escrita comunica quietly com poder. Profundidade elegante.

## 2. Caráter (Os Filtros Éticos)
Para líderes de mercado (médicos, advogados), reputação precede a receita. O sistema é moldado para carregar um forte filtro ético anti-venda desesperada, anti-manipulação e anti-clickbait.
- Toda geração textual deve focar num princípio educativo genuíno.
- Alegações absurdas sem prova devem ser flagradas no _Quality Checklist_ pelo sistema/agente ou humano.
- O sistema trabalha não com hipérbole ("Seja milionário amanhã"), mas com realismo sofisticado ("Os gargalos operacionais que devoram até 14% do equity da sua clínica").

## 3. Cristo no Centro (O Propósito Maior)
Uma crença essencial incorporada no núcleo e governança do projeto liderado sob a perspectiva dos princípios do Grupo Torq. 
- Significa que todo trabalho produzido possui um aspecto de integridade intrínseco. 
- Servir bem acima da escalabilidade predatória. 
- Tratar a verdade como base intocável durante o processo criativo. 
O produto, suas comunicações, e os fundadores servem para honrar excelência e a verdade, construindo uma marca que suporta o teste do tempo guiada pela retidão.
