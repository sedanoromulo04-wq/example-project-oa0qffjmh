# Torq OS - Design Later

Este diretorio guarda a frente de design do sistema, mas fora da urgencia atual.

## Papel desta pasta

Design continua importante, mas nesta etapa ele entra como camada posterior.
O foco agora e estruturar memoria, pesquisa, estrategia, conversao,
orquestracao e governanca. Quando essa base estiver estavel, esta pasta vira a
frente oficial de experiencia e interface.

## O que pode entrar aqui depois

- arquitetura de navegacao visual
- wireframes das telas principais
- sistema visual do Torq OS
- componentes e estados de interface
- direcao de UX para operacao interna
- refinamento estetico e consistencia visual
- prompts de design para ferramentas como Google Stitch
