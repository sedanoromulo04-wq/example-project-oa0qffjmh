# Torq OS - Main Idea

Este diretório guarda a ideia central do sistema.

## Proposito

Torq OS existe para materializar a mente do founder, organizar memoria, orientar
estrategia, estruturar conversao e só depois derivar conteudo e distribuicao.
O sistema nao nasce como um gerador de posts. Ele nasce como um backoffice
operacional com foco em fidelidade intelectual e decisao clara.

## O que entra aqui

- tese principal do sistema
- principios inegociaveis
- definicao do que Torq OS faz e do que nao faz
- mapa de modulos do sistema
- regras de priorizacao

## Conteudo inicial

- V1 e interno, com HITL forte
- estrategia vem antes de execucao
- memoria vem antes de volume
- conteudo e derivacao, nao finalidade
- design fica para uma fase posterior
